RimTUB - RimMirK's Telegram User Bot

RimTUB ‚Äî is a powerful and convenient open-source Telegram userbot that's easy to adapt to your needs.

Everything necessary is already built-in, and extending its capabilities only requires installing the modules you need. Simple to launch, flexible in configuration, and completely under your control ‚Äî RimTUB is created to do more with less effort.

ü§ñ UserBots are scripts that work on behalf of your personal Telegram account. They can do everything a regular user can do.

Useful "out of the box" commands

  • .me ‚Äî info about the userbot (version, uptime, etc.).
  • .help ‚Äî help on userbot.
  • .ping ‚Äî ping test.
  • .restart ‚Äî restart the userbot.
  • .calc <expression> ‚Äî calculator.
  • .tr <language> <text/reply/quote> ‚Äî translator.
  • .dml <link> ‚Äî install module from link (from website, for example).
  • .dmf <reply to file> ‚Äî install module from file.

Installation

Windows

Step 1. Download RimTUB

  1. Go to: GitHub Releases
  2. Click on the item with the Latest badge (this is the latest version).
  3. Below, click on assets (file list)
  4. In the file list, find an archive named like RimTUB-XXX.zip (XXX is the userbot version) ‚Äî click on it to download.
  5. When it's downloaded ‚Äî open the folder with the file, right-click on the archive and select "Extract All".
  6. Enter a path, for example: C:\RimTUB, and click "Extract".

Step 2. Create a Telegram bot

  1. Open Telegram and find user @BotFather.
  2. Click Start or type /start if the bot is silent.
  3. Type /newbot, set a name and link for the bot (for example, RimTUB_nickname_bot).
  4. BotFather will send you a long token ‚Äî copy it (it looks like 123456:ABC-DEF...).
  5. ‚ùóÔ∏è Type /setinline, select your bot, and type any text, for example asdfjwekjdsf ‚ùóÔ∏è

Step 3. Configure RimTUB

  1. Go to the C:\RimTUB folder that you just unpacked.
  2. Find the config.yaml file there.
  3. Open it with a double click. If nothing happens ‚Äî right-click and select "Open with ‚Üí Notepad".
  4. Insert your data there. Example: 
    PHONES:
  - +12345678990 # Your phone number linked to Telegram
  - +380XXXXXXXX # You can add multiple accounts
BOT_TOKEN: 123456:ABC-DEF...  # Token provided by BotFather
  5. Save the file: File ‚Üí Save.

Step 4. Run RimTUB

  1. Go to python.org.
  2. Hover over "Downloads" and select Windows.
  3. Click "Download Python 3.11.9".
  4. When the installer downloads ‚Äî MAKE SURE to check "Add Python to PATH", then click "Install Now".
  5. Wait for the installation to complete and close the window.

Step 5. Install dependencies

  1. Press Win + R keys, a window will appear.
  2. Type cmd and press Enter ‚Äî a black window will open (command prompt).
  3. Enter the following commands one by one (press Enter after each): 
    cd C:\RimTUB
python -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
python main.py
  4. ‚ö† If a window appears asking for permission to access the internet ‚Äî click "Allow".

Step 6. Confirm login

  1. After launching, the bot will ask you to enter a code.
  2. Telegram will send you an SMS ‚Äî enter this code in the console.
  3. If you have two-factor authentication enabled (password when logging into Telegram) ‚Äî enter it as well. You'll only need to do this once

üéâ Done! RimTUB is running! Hooray!
