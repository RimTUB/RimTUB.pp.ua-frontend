# тут сокращенная версия ее можно не читать — вся основа в readme.md


npm i — скачать все зависимости

----------------------------------------------------------------------

gulp — в режиме development
gulp --build — собрать сборку в режиме production
gulp --prev — посмотреть какой результат вышел в режиме production
npm run validate — валидация для хтмл (не нужно так как при каждом изменении html будет валидация)
npm run stylelint — проверка на порядок написания стилей (не нужно, если у вас есть плагин и запись в settings.json)
npm run deploy — создает ветку 'gh-pages' в вашем репозитории и публикует проект на github pages. Пример ссылки - `https://<username>.github.io/<repo-name>/`

----------------------------------------------------------------------

└─app # главная папка для работы
	├───blocks # для html файлов
	├───fonts # для шрифтов
	├───images # для хранения картинок
	│   └───sprite # если вам нужно подключить SVG через sprite, все SVG-файлы помещаются сюда. Если не поместить svg в эту папку, то sprite сделан не будет
	├───js # скрипты
	│   ├───components # для js-компонентов
	│   └───libs # для библиотек
	├───resources # для хранения разных файлов(php, favicon, video)
	└───scss # для стилей
			├───base # для базовых стилей(переменные, глобальные стили, normalize)
			├───components # для scss-компонентов
			├───libs # для библиотек
			└───mixins # миксины для проекта

│───  .editorconfig # файл с настройками форматирования кода
│───  .eslintrc.json # для настроек js кода
│───  .htmlhintrc # файл с настройками htmlhint
│───  .stylelintrc.json # файл с настройками stylelint
│───  gulpfile.js # сборка
│───  package.json # файл с настройками сборки и установленными пакетами
│───  readme.txt # документация сборки

----------------------------------------------------------------------

Для подключения всех html-компонентов в index.html используйте динамический подход
@include('./blocks/filename.html')

----------------------------------------------------------------------

fonts — туда шрифты, любого формата кроме woff, они не будут конвертированы;
файл шрифтов должен быть через тире "Roboto-Medium", "Roboto-BlackItalic", тогда автоподключение шрифтов будет работать корректно

app/scss/_fonts.scss — создаться когда автоматически подключаться шрифты, если вы добавили какой-то шрифт, то нужно подключить его самому в этом файле. Либо удалить этот файл и перезапустить сборку, тогда шрифты сами подключаться в этот файл

----------------------------------------------------------------------

чтобы не писать долго путь, можно прописать в настройках vs code
"path-autocomplete.pathMappings": {
  "#img": "${folder}/app/images"
}
для этого скачайте плагин Path Autocomplete и запишете в json параметры код выше
теперь можно можно писать #img и оно автоматически подскажет вам путь к картинке

----------------------------------------------------------------------

в js можно использовать es6 импорты. Пример: import './components/test.js';
Так же подключать css, примпер 'import simplebar/dist/simplebar.css'
